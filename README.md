# 🤖 FlowMind AI

> Enterprise AI Automation Platform built with Python, Streamlit, LangChain, Google Gemini, ChromaDB, and SQLite.

---

# 🚀 Overview

FlowMind AI is a modular AI Automation Platform that combines multiple AI agents into a single application.

The platform can:

- Analyze resumes
- Chat with PDF documents
- Perform AI research
- Generate reports
- Track activities
- Show analytics
- Manage authentication
- Store data in SQLite
